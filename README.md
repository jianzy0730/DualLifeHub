# 双人生活簿 / Dual Life Hub

面向 Operit 的双人生活记录工具包。财务和备忘默认支持用户与 AI 两方，生理期记录可在首次配置时选择关闭、单人或双方模式。

## 功能

- 双人财务：我的账、AI 钱包、双方转账、分类统计与流水
- 双人备忘：我的备忘与 AI 的备忘
- 生理期日历：可关闭、仅记录一方或双方记录，不做日期预测
- 双主题色：用户与 AI 分别配置颜色，身份卡片、按钮、标记和主要财务卡会同步变化
- 自定义称呼：设置 AI 如何称呼用户，以及用户如何称呼 AI

## 安装与初始化

1. 在 Operit 中导入并启用 `.toolpkg`。
2. 打开工具箱中的“生活簿配置”。
3. 设置双方称呼、生理期记录模式和两套主题色。
4. 从侧栏进入“生活簿”。

首次未完成配置时，主页面会先显示初始化页面。

## 生理期模式

- 不启用
- 仅记录用户
- 仅记录 AI
- 双方记录

界面会根据模式自动隐藏无关入口、筛选与第二个人的状态卡。

## 主题色说明

双方颜色独立应用于主要财务卡、周期状态卡、备忘归属标记、日历标记、按钮和筛选状态。普通内容卡保持中性色，圆饼图继续使用分类颜色，以免混淆数据类别。

## 数据与隐私

数据保存在本地工具包运行目录中。公共版使用独立包 ID：`com.community.dual_life_hub`。

## 包内结构

- `manifest.json`：包信息与资源声明
- `dist/`：ToolPkg 主脚本、工具与 UI 入口
- `resources/runtime/server.py`：本地数据服务
- `resources/webapp/index.html`：生活簿界面
